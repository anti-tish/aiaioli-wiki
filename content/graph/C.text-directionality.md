# Text Directionality

`Concept` · `vector_semantics` · *hypothesis* · visibility: **publishable**

All written text is produced with a direction in mind — even a note to yourself, even a prayer. A text requires two distinct, asymmetric acts to exist: production and consumption. Directionality is therefore a SPATIAL property of language — meaning is a path traversed, not only a position occupied — and it is absorbed into training as an asymmetry the model never sees as a choice.
