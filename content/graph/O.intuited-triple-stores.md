# Noun, Verb, Noun

`Observation` · `knowledge_representation` · *personal_truth* · register: **public** · auto-extracted, not yet author-verified

If [[C.relational-ontology]] is the claim — structure establishes meaning — the triple is its
smallest working unit. Noun, verb, noun: two things and the relation between them. *Loreal
acquires Armani Beauty* — the verb *acquires* implies a company, so both names now sit in the
company graph. *Germany invades Poland* — *invades* implies territory, a different kind of
edge entirely. The verb doesn't just link the nouns; it selects which attributes they bring with
them. Change the verb and the whole neighborhood around each noun changes.

This is the RDF triple — subject, predicate, object — the atom the semantic web is built from. It
turned up here in a note written before that name was in reach: the structure arrived first, the
vocabulary later ([[P.reason-before-name]]). Which is the point worth keeping, and it is
about the idea rather than the arriver — a relational model of knowledge isn't exotic or
technical by nature. Its base unit is a sentence a child can parse. The exotic part was ever only
the name.
