# RAG (Retrieval-Augmented Generation)

`Concept` · `knowledge_systems` · *verified_fact* · visibility: **publishable**

Giving a model a fixed reference point — an external document or knowledge base — that acts as an anchor during generation. Doesn't fix the projection problem but gives the model something to be wrong *relative to*, which is a meaningful practical improvement. Error becomes measurable. Her interest: not factual grounding but what connections and meanings emerge relative to the anchor. Wikidata as RAG anchor = personal observations interpreted relative to shared external reference point.
