# Knowing Before the Name

`Connection` · `cognition` · *hypothesis* · visibility: **publishable**

**From:** [[P.hypervigilance-as-semiotics]]
**To:** [[P.reason-before-name]]
**Relationship:** is_structurally_identical_to

P.reason-before-name: the reason arrives fully formed before the name does; she knows a thing is true before she can say why. P.hypervigilance-as-semiotics: excessive semiotic pattern recognition — she's already read the room, catalogued the signals, and assessed the situation before conscious processing catches up. Both: the fast system completes before the slow system can report on it. Pre-verbal knowing (reason-before-name) and pre-conscious pattern detection (hypervigilance) are the same cognitive architecture at different timescales and different domains of application. The reframe — from 'disordered' to 'observational' — is also the same move: in both cases, the speed is the feature, not the bug. The problem isn't that the system fires early; the problem is whether you can tell the alarm from the observation.
