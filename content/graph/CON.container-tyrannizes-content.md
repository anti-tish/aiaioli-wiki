# Container Tyrannizes Content

`Connection` · `epistemology` · *hypothesis* · visibility: **publishable**

**From:** [[O.coordinate-tyranny]]
**To:** [[O.trapped-in-chats]]
**Relationship:** is_structurally_identical_to

GIS assumes physical adjacency is the meaningful relationship; the chat interface assumes linear conversation is the meaningful structure for writing. In both cases the content already exists — the ideas, the writing — and the problem is that it's in the wrong container. The coordinate model excludes horror-proximity, flight-proximity, cultural-proximity. The chat model excludes indexability, discoverability, the writing's own legibility to itself. Same structure: a container that models only the proximity it was designed for, treating that model as exhaustive. The coordinate is to geographic space as the chat thread is to ideas: a correct but insufficient container that becomes tyrannous when you have no other option.
