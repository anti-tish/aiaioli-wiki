# Text Georectification (prepositions as control points)

`Question` · `geography_cartography` · *hypothesis* · visibility: **publishable** · extraction: **auto**

Proposal (Tish): could embedded directionality and positionality act as control points for 'rubbersheeting' a text raster into spatial-semantic coherence? (The GIS vocabulary — datum, rubbersheeting, georectification, control points, raster — is Tish's own; the linguistics scaffolding below is Claude-introduced.)

Refinement worth testing: prepositions are *relations* (the warp/displacement field), not points. The true control point is the deictic center — the I-here-now origin — which linguistics already names Bühler's origo and GIS already names the datum. Marrying origo ↔ datum is the move. Existing scaffolding: Levinson's frames of reference (intrinsic/relative/absolute).

The honest strain (Tish flagged 'not quite isomorphic'): georectification needs external ground truth — the Earth. Semantic space has none (the projection problem). So a text raster rectifies to RELATIVE coherence anchored on its origo, never to ABSOLUTE position. The analogy breaks exactly where 'no view from nowhere' predicts. All text is georeferenced to a situated origo, never to a god's-eye datum. Geography's error is pretending the absolute datum exists; there is only ever an origo.
