# Shipping It as Software Shows You Didn't Think

`Concept` · `hci` · *unverified_claim* · visibility: **publishable**

The manifesto stance, deliberately inflammatory: 'I'm not going to download your software because the fact that you released it as software shows me you're not thinking about this enough.' Underneath the provocation is a real open question she didn't resolve — should a thing built with Claude ship as an executable, or as 'I told Claude what to do, and you can too'? A categorical rejection of the current state of HCI, framed as a refusal, not a feature request.
