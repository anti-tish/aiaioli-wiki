# Capture Tools Fail

`Pattern` · `knowledge_systems` · *personal_truth* · visibility: **publishable**

Capture tools fail because they require structure before thought. Every Airtable attempt, every structured Obsidian attempt, the stoner notes working better than everything else — same mechanism. Structure imposed before capture rejects anything that doesn't conform to it. The tool that asks you to categorize before you finish the sentence is the tool that will never get the good stuff. The good stuff arrives before it knows what it is. This explains every PKM failure mode simultaneously. The solution isn't a better capture tool — it's relocating the structuring step to after capture (see P.corpus-plus-dialogue, P.system-must-fit-not-teach).
