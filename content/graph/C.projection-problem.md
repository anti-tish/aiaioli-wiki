# The Projection Problem (NLP)

`Concept` · `vector_semantics` · *hypothesis* · visibility: **publishable**

Embedding spaces aren't neutral geometry — they're projections encoding whose language is centered. NLP mostly does spherical geometry (space as neutral); the honest move is geodesy (space as constructed). You can't measure the distortion without an external referent, and there is none. Hallucination is unsolvable the way map distortion is: you pick what you're willing to lie about.
