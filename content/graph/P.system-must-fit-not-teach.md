# System Must Fit, Not Teach

`Pattern` · `cognition` · *personal_truth* · visibility: **publishable**

'The system has to match who you are, not teach you to be someone else.' She became a hoarder in a city with strict bottle-return recycling rules — the system required a behavior she couldn't sustain, so it generated exactly the problem it was trying to solve. The constraint-vs-fit distinction: a system that requires you to change your cognitive shape will be sabotaged, not adopted. The tool has to work with her actual operating mode (loose plans, some impulsivity) or it creates the dysfunction it was supposed to prevent.
