# Geographic Knowledge Is Never Neutral

`Pattern` · `geography` · *confident* · visibility: **publishable**

Geographic knowledge encodes the priorities of whoever held the instruments. The map is not a view from nowhere. The projection choice, what gets named, what gets erased: all of this is a decision that presents itself as geometry. Instances: missionary cartography (Jesuits mapping Ethiopia for reconnaissance and resource extraction), Vatican's determination of what is 'mappable,' Harley's critical cartography, embedding space as projection with no neutral view from nowhere. The same pattern as C.projection-problem in NLP: the mapmaker is never absent from the map, the model trainer is never absent from the embedding space. Harley's argument about cartography IS the same argument about NLP — not analogy, structural identity.
